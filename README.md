## 实现功能
* 防止微信好友撤回聊天消息
* 防止微信好友删除朋友圈动态、评论

目前支持最新微信 6.5.13 版。本插件使用静态/动态分析来智能比对函数签名，几乎不会因为微信更新版本而失效。
